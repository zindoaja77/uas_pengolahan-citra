# Biker OCR Flask Application

Aplikasi OCR dengan tema biker/motor yang berjalan di Flask.

## Fitur
- Upload gambar dari komputer
- Capture gambar dari kamera
- Ekstrak teks dengan Tesseract.js
- Terjemahan teks
- Text-to-speech
- Download hasil
- Tampilan dengan tema biker
